<i>Минимальные требования для работы плагина</i>:
<br />
Версия движка Moguta.CMS начиная с <span style="color:red">8.5.0</span>
<br />
<br />
<span style="color:red">Внимание! Фильтрация по брендам доступна только на редакциях "Маркет" и "Гипермаркет" в связи с отсутствием на редакции "Минимаркет" характеристик товаров</span>
<br />
<br />
Плагин добавляет новую характеристику для товаров под название "Бренд", для каждого бренда создает свою страницу, добавляет соответствующее поле в фильтр каталога и выводит карусель брендов по шорткоду [mg-brand].
<br />
<br />
<h2>Первый запуск</h2>
<ol>
<li><b>Добавьте новые бренды для товаров.</b> Для добавления бренда нажмите кнопку <i>"Добавить бренд"</i> и в появившемся модальном окне задайте название бренда, его лого, описание и seo-свойства для карточки бренда.</li>
<li><b>Назначьте нужным товарам их бренды.</b> Перейдите в редактирование нужного товара и заполните для него поле <i>"Бренд"</i>.</li>
<li><b>Добавьте шорткод на сайт.</b> По шорткоду <strong>[mg-brand]</strong> выполняется вывод карусели брендов. Его конфигурацию можно произвести на странице плагина в разделе <i>"Настройка карусели брендов"</i>. В некоторых шаблонах шорткод может быть уже вставлен.</li>
</ol>
<b>Замена старого плагина "Логотипы брендов".</b> Для замены старого плагина переименуйте шорткод <strong>[brand]</strong> на <strong>[mg-brand]</strong>. В некоторых шаблонах кроме самого шорткода, присутствует проверка на включённость плагина. Выглядит это примерно так:
<br>
<blockquote>
&lt;?php if (class_exists('brand')): ?&gt; <br>
            .... <br>
            [brand] <br>
            .... <br>
&lt;?php endif; ?&gt;
</blockquote>
<br>
В таком случае нужно не только заменить <strong>[brand]</strong> на <strong>[mg-brand]</strong>, но и заменить class_exists('brand') на class_exists('Brands');
<br />
<br />
<h2>Синхронизация</h2>
<br />
В плагине предусмотрена синхронизация с характеристиками. Это необходимо для тех случаев, если вдруг одно из свойств было потеряно или отвязано от бренда. 
<ul>
    <li>Если по каким-то причинам было потеряно свойство, привязанное к бренду, запись бренда на странице настроек плагина перекрасится в красный. Для того, чтобы создать новое свойство на основе бренда, нажмите кнопку <i class="fa fa-refresh"></i></li>
    <li>Если по каким-то причинам был потерян бренд, но свойство существует, запись бренда на странице настроек плагина перекрасится в желтый. Для того, чтобы создать новый бренд на основе свойства, нажмите кнопку <i class="fa fa-plus"></i>. После добавления бренд будет без лого и описания, поэтому не забудьте отредактировать его. </li>
</ul>
<br />
Данные опции крайне необходимы, если вы решили переключить характеристику в настройках плагина для создания брендов на основе уже существующих свойств.
<br />
<i>В случае потери всей характеристики на странице настроек плагина необходимо сгенерировать новую характеристику или выбрать существующую</i>
