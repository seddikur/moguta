<h2>О плагине</h2>
<p>Плагин «Повтор заказа» выводит две кнопки в личном кабинете в разделе заказов для полного копирования заказа (товары, способ оплаты, способ доставки, данные о получателе) и для копирования содержимого заказа в корзину. Если каких-то товаров не хватает или они неактивны пользователь увидит сообщение об этом.</p>

<p><img src="../mg-plugins/mg-repeat-order/images/repeat.png" alt="Кнопки для повтора заказа"></p>

<h2>Установка плагина</h2>
<p>Скопируйте код:</p>
<div class="highlight">
<pre>
<code>
&lt;?php if (class_exists("repeatOrder")): ?&gt;
  [repeat-order id=&lt;?php echo $order['id']?&gt;]
  [edit-order id=&lt;?php echo $order['id']?&gt;]
&lt;?php endif; ?&gt;</code>
</pre>
</div>
<p>Вставьте скопированный код в файл вашего шаблона отвечающего за вывод заказов в личном кабинете пользователй.</p>
<p>В стандартном шаблоне это компонет personal</p>
