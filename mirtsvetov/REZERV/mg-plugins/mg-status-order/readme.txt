<h3>О плагине</h3>

<p>Плагин позволяет создавать и редактировать статусы заказов.</p>

<p>После создания новых статусов, сделайте статусы видными для администраторов.</p>

<ol>
    <li>
        Пользователи -> группы пользователей
    </li>
     <li>
        В пункте "Вы редактируете" выберите нужную вам группу
    </li>
     <li>
        В "Доступных статусах" отметьте те которые вы хотели бы сделать видимыми
    </li>
</ol>
